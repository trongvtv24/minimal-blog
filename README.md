# Minimal Focus Blog

A minimal, high-performance blog built with Next.js, focused on readability and balanced design.

## Features
- **Modern Tech Stack:** Next.js 15+ (App Router), TypeScript, Tailwind CSS 4.
- **Content:** MDX-based posts with frontmatter support.
- **Search:** Built-in instant search functionality.
- **Readability:** Optimized typography (Inter) and layout (720px content width).
- **Engagement:** Integrated Facebook Share and native Facebook Comments.

## Tech Stack
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [next-mdx-remote](https://github.com/hashicorp/next-mdx-remote)
- [Gray-Matter](https://github.com/jonschlinkert/gray-matter)

## Getting Started
1. Clone the repository: `git clone https://github.com/trongvtv24/minimal-blog.git`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

## Deployment
Deployed on GitHub: [minimal-blog](https://github.com/trongvtv24/minimal-blog)
