## Font chữ quan trọng hơn bạn nghĩ

Typography chiếm hơn 90% nội dung trên web. Vậy mà nhiều developer vẫn dùng font mặc định và line-height 1.5 mà không suy nghĩ gì thêm.

Một hệ thống typography tốt giúp:
- Người đọc không bị mỏi mắt
- Hierarchy thông tin rõ ràng
- Cảm giác chuyên nghiệp, đáng tin

## Chọn font

### Nguyên tắc

1. **Tối đa 2 font families** cho toàn site
2. Ưu tiên font có nhiều weight (400, 500, 600, 700)
3. Kiểm tra readability ở font size nhỏ (14px)

### Gợi ý

- **Body**: Inter, Source Sans Pro, Noto Sans
- **Heading**: cùng font nhưng weight khác, hoặc serif như Lora

## Type Scale

Đừng tự chọn font size tùy hứng. Dùng scale có hệ thống:

| Element | Size | Line Height |
|---------|------|-------------|
| Body | 18px | 1.75 |
| H1 | 36px | 1.2 |
| H2 | 28px | 1.3 |
| H3 | 22px | 1.3 |
| Small/Meta | 14px | 1.4 |

## Line Height & Measure

- **Body text**: line-height 1.6–1.8 cho tiếng Việt (cần nhiều hơn tiếng Anh)
- **Measure** (chiều rộng dòng): lý tưởng 60-75 ký tự, tương đương ~720px với font 18px
- **Heading**: line-height thấp hơn body (1.2-1.3) vì heading ngắn

## Khoảng cách giữa các đoạn

Paragraph spacing nên bằng line-height hoặc hơn một chút. Với body 18px, line-height 1.75, spacing đoạn nên khoảng 24-32px.

> Typography tốt là typography bạn không nhận ra — nó chỉ đơn giản khiến bạn đọc dễ dàng hơn.
